
---

## 📁 Project Structure

```
project/
├── client/      # Frontend (React + Vite + Tailwind)
└── server/      # Backend (Node.js + Express)
```

---

## 🛠️ Tech Stack

### Frontend

* React
* React Router DOM
* Axios
* Tailwind CSS
* Framer Motion
* Vite

### Backend

* Node.js
* Express.js
* MySQL
* Sequelize / Prisma
* JWT (Authentication)
* Multer
* Bcrypt
* CORS
* Dotenv

---

---

## ▶️ Run Frontend

```
cd client
npm install
npm run dev
```





## ▶️ Run Backend

```
cd server
npm install
npm run dev
```
