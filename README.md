
---

## 📁 Project Structure

```
project/
├── client/      # Frontend (React + Vite + Tailwind)
└── server/      # Backend (Node.js + Express)
```

---

## 🛠️ Tech Stack

### Frontend

* React
* React Router DOM
* Axios
* Tailwind CSS
* Framer Motion
* Vite

### Backend

* Node.js
* Express.js
* MySQL
* Sequelize / Prisma
* JWT (Authentication)
* Multer
* Bcrypt
* CORS
* Dotenv

---

## ⚡ Installation

### 1. Clone the repository

```
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

---

## ▶️ Run Frontend

```
cd client
npm install
npm run dev
```

Runs on:

```
http://localhost:5173
```

---

## ▶️ Run Backend

```
cd server
npm install
npm run dev
```

Runs on:

```
http://localhost:5000
```
